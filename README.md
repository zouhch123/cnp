# 网络程序设计

## 1. 概述

本工程包含了“网络程序设计”课程使用的各类相关材料，包括课件、实验指导书、示例代码等。

## 2. 目录结构

项目根目录下的目录功能如下：

* presentations：课件幻灯片
* fugures：用到的图片
* setup：beamer模板
* assignment：实验指导书
